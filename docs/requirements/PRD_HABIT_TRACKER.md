# HabitTracker - 习惯打卡

**版本**: v1.0 (MVP)  
**创建日期**: 2026-03-03  
**负责人**: pm

---

## 1. 产品概述

### 1.1 产品愿景
一款习惯养成打卡应用，帮助用户坚持好习惯，戒除坏习惯。

### 1.2 目标用户
- 18-40 岁，想自我提升
- 想养成好习惯
- 想戒除坏习惯

### 1.3 核心价值
- **习惯追踪**: 每天打卡记录
- **成就系统**: 激励坚持
- **数据统计**: 看到进步

## 2. 功能需求

### 2.1 习惯管理 (P0)
| 需求 ID | 描述 | 优先级 |
|---------|------|--------|
| HABIT-001 | 创建习惯 (名称/图标) | P0 |

| HABIT-003 | 习惯分类 | P1 |
